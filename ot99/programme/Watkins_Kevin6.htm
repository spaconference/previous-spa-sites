ORBs and ODBMSs: A Marriage Made in Heaven?

Wednesday 11:45   Tutorial 75 minutes

The problems and pitfalls of integrating Object Databases with Object Request Brokers. Strategies that work well, and those that don't work quite so well.

Kevin Watkins

It is common these days to find multi-tier applications featuring an Object Database as the backend repository with CORBA ORB servers providing access to the middle-tier business logic. Such an architecture can result in high performance, scalable systems but the way that CORBA servers and persistance co-exist needs to be thought out very carefully. This tutorial looks at some strategies, idetifying those that work well and those that pose problems. Concrete examples of the principles covered are shown with reference to deployed systems.

Kevin Watkins (kwatkins@objectdesign.com)

Kevin Watkins is a consultant with Object Database vendor

Object Design (http://www.objectdesign.com). Kevin has been

involved with the design of several systems accross a broad

spectrum of application domains.
