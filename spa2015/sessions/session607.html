Software Practice Advancement Conference

SPA Conference session: Engaging young people in software development

One-line description:An update on how the youth are learning technology and a workshop on the tools they are using
 
Session format: Talk and Workshop - 75 Mins [read about the different session types]
 
Abstract:In September 2014 Computing was made a "proper subject" in schools across the country meaning children as young as 4 are learning computing skills in the classroom. Programmes like the Young Rewired State Festival of Code mean that young people are leaving school with a ready made network of tech industry contacts. This workshop aims to show you what it is like to be studying Computing at school as a young person today, inform you about the places you can support and nurture the nation's youth, from all walks of life, in their coding endeavours by getting some hands on development with some of the most popular tools - AppInventor and Ratchet (no Scratch however!).
 
Audience background:Basic programming and HTML knowledge is a plus but I would advise anyone interested to attend regardless of their level - AppInventor is designed for complete beginners.
 
Benefits of participating:- Learn how to teach young people to code
- An update on what it is like to be a young person learning about technology in 2015
- Learn about where you can support young people learning to code
- Learn about the products and resources for teaching young people how to code
 
Materials provided:
 
Process:I will give a talk sharing my knowledge of teaching young people to code in and out of school and then lead a workshop using App Inventor and Ratchet.
 
Detailed timetable:00:00 - 00:30 - Talk about young people learning to code and how people can get involved.
00:30 - 00:35 - Introduction to AppInventor and Ratchet
00:35 - 00:55 - Work on App using Technology of choice
00:55 - 01:00 - Discuss any problems or positive points about the software
01:00 - 01:05 - Add finishing touches to apps
01:05 - 01:15 - 30 second lightning pitches demoing apps to the group.
 
Outputs:Make a fully functioning android app or web based app.
Publicly available slides.
Worksheets introducing AppInventor and Ratchet
 
History:I have presented a very similar session at The CAS #include diversity conference 2014 with a focus on helping young women in tha classroom.
