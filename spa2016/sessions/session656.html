SPA Conference session: Play the GetKanban Game

One-line description:Experience life in a Kanban team with this this popular board game
 
Session format: Long Tutorial [read about the different session types]
 
Abstract:Do you wonder what Kanban looks like in real life? Come play this game and experience it!
From their website: "The getKanban Board Game is a physical game designed to teach the concepts and mechanics of Kanban for software development in a class or workshop setting. It's hands-on, engaging, and fun!"
 
Audience background:A little knowledge of Kanban helps but is not strictly necessary
 
Benefits of participating:Participants will simulate working in a Kanban team, experience day to day problems and learn how Kanban can help them
 
Materials provided:I have one game kit (GetKanban v4)
 
Process:Attendance is limited to 6 people per game kit (we could stretch it to 7-8 if we really want). I have one kit. If we find more kits I can run it with multiple teams at the same time. Maybe some of the other speakers have some?
 
Detailed timetable:00:00 - 00:30 Kanban Introduction
00:30 - 00:50 game instructions
00:50 - 04:00 GetKanban game (with breaks every now and then)
04:00 - 04:15 debrief, learning points
 
Outputs:List of the learning points
 
History:I've facilitated the game many times internally at my company
