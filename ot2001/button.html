To put this button on your web page...

In html ---

Insert these lines: 
   <a href=http://www.ot2001.org/><img target=_top
      border=0 src=http://www.ot2001.org/ot-button.gif 
      width=100 height=100></A>

Using a page composer ---

  • Edit your web page
  • Drag the image into the page
  • Link the image to http://www.ot2001.org
  • Publish page (with copy of image)
