SPA Conference session: Distributed applications on the Sun SPOT

One-line description:Create an application that can be used during the conference
 
 Session materials
 
Session format: Working group [read about the different session types]
 
Abstract:The Sun SPOT is a small low-cost battery-powered computing device that can be connected to a variety of transducers and can communicate wirelessly with similar devices. It runs a Java operating system, and both device drivers and applications are written in Java. This is unusual - software for these kinds of devices is typically written in C or C-like languages.

The nature of these devices opens up a whole new field of possible applications. The purpose of this session is to expose participants to the possibilities, and give them the opportunity of building applications that explore this space.

Working in groups, participants will create versions of an application that can be tested during the remainder of the conference. The exact nature of this application is to be decided, but one possibility would be a system for gauging opinion during keynotes or other sessions.
 
Audience background:- Must know Java
 
Benefits of participating:- Learn about the possibilities for a new generation of computing devices
- Get specific experience of developing for the Sun SPOT
 
Materials provided:We'll provide at least enough SPOTs for groups to develop and test their applications. We may even be able to provide enough SPOTs for every conference participant to carry one during the event.

Participants will need to bring laptops. We'll provide the development tools, although we'd like participants to pre-install some stuff to save time.
