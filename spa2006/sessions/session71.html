SPA Conference session: A New Face for Java

One-line description:A detailed introduction to JavaServer Faces
 
 Session materials
 
Session format: Tutorial [read about the different session types]
 
Abstract:At present JavaServer Faces (JSF) is one of the hottest topics in the Java Enterprise Edition (JEE) community. As a new technology, JSF represents a radical rethink in the way that user interfaces for Web-deployed applications are developed and enables richer, more complex application interfaces to be built without raising the bar of complexity.

This session will provide a comprehensive introduction to JSF, sprinkled with live demonstrations, encompassing both the tasks of basic development along with the internals and extension of the framework. We will also look at (and demonstrate) the potential for the deployment of JSF applications into devices other than standard web browsers opening up a huge potential for novel application of the technology.
 
Audience background:Java Enterprise Edition developers
 
Benefits of participating:* Gain an in-depth introduction to JSF
* Understand the difference between JSF and other Web UI technologies
* Experience some of the potential for multi-channel development with JSF
