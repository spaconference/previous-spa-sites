SPA Conference session: A practical introdution to Ruby on Rails (RoR)

One-line description:Hands on development of a small application in Rails, the popular Web Application framework for the Ruby language.
 
 Session materials
 
Session format: Workshop [read about the different session types]
 
Abstract:Ruby is an elegant, dynamically typed Object-Oriented scripting language that has steadily gained in popularity over the last 10 years. Rails is an open-source Agile Web Application Framework built on top of Ruby. Rails emphasizes convention over configuration and strives for simplicity and the DRY principle (Don't Repeat Yourself). It has attracted much interest over the last year, and has done much to promote the Ruby language itself.

Rails is based on a Model-View-Controller paradigm and incorporates a number of design patterns, such as ActiveRecord for persistence, which simplify common development tasks. Rails also has strong support for AJAX (Asynchronous Javascript and XmlHttpRequest), a popular way of developing next-generation browser based interfaces as evinced by Google Maps and the like. Rails is however more than just a development framework as it includes many useful scripts for the creation of new components, a complete testing framework and a build language.

Extravagant claims have been made for productivity gains using Rails - some say it is ten times faster than traditional development techniques. There is only one way to find out - try it for yourselves!
