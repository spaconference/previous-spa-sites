SPA Conference session: Java Scripting

One-line description:Where JVM-based scripting fits into a project.
 
 Session materials
 
Session format: Tutorial [read about the different session types]
 
Abstract:There are now over 20 languages targetting the Java Virtual Machine. Most are more dynamic than Java, and provide opportunities to escape the straightjacket of their host.

We start by looking at some of the languages available (Groovy, Jython, Beanshell etc), and then go on to examine where they can be used in a typical project, with code examples.

We'll leave plenty of time at the end to catalogue your experiences.
 
Audience background:Pretty much the Java mob, although dynamic language bigots may apply.
 
Benefits of participating:* Hit the ground running
* Cross-pollinate
* Lose weight
